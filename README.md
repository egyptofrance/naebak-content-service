## 📝 الوصف

خدمة إدارة المحتوى الثابت في التطبيق، مثل صفحات "عن التطبيق" و "سياسة الخصوصية" و "شروط الاستخدام".

---

## ✨ الميزات الرئيسية

- **إدارة المحتوى الثابت**: إضافة وتعديل المحتوى الثابت.
- **دعم متعدد اللغات**: إمكانية إضافة المحتوى بلغات متعددة.
